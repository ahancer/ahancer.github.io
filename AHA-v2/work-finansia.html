Finansia Hero - Stock Brokerage

Making stock trading easy

The Problem

While Finansia Hero app has many powerful tools that can help users trade such as automated trading, its complexity and UI confused many users.

The Goal

How might we redesign the new app to be as simple and intuitive as possible while still keeping all the relevant data accessible to the user when they need it most?

Our Process

Fuelled by user research, our design process blends design thinking, lean UX methodology, and agile to design and iterate solutions that solve real user problems.
